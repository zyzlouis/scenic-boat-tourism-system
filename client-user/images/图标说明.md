# TabBar 图标说明

## 需要的图标

| 文件名 | 说明 | 尺寸要求 |
|--------|------|----------|
| home.png | 首页图标（未选中） | 81px × 81px |
| home-active.png | 首页图标（选中） | 81px × 81px |
| order.png | 订单图标（未选中） | 81px × 81px |
| order-active.png | 订单图标（选中） | 81px × 81px |

## 图标要求

- **格式**：PNG 格式
- **大小**：不超过 40KB
- **尺寸**：建议 81px × 81px
- **颜色**：
  - 未选中状态：灰色 (#666666)
  - 选中状态：蓝色 (#1989fa)
- **背景**：透明背景

## 获取图标的方式

### 方式一：在线图标库（推荐）

1. **Iconfont（阿里巴巴图标库）**
   - 网址：https://www.iconfont.cn/
   - 搜索关键词："home"、"order"
   - 下载 PNG 格式，尺寸选择 81px

2. **IconPark（字节跳动图标库）**
   - 网址：https://iconpark.oceanengine.com/
   - 下载后改成对应的颜色

### 方式二：使用设计工具创建

1. **Figma / Sketch / AI**
   - 创建 81×81 画布
   - 绘制简单的图标
   - 导出 PNG（2x）

### 方式三：使用我提供的临时图标

我已经为您准备了一套简单的占位图标（见下方）。

## 快速测试方案

**当前已暂时移除 tabBar**，可以直接编译测试核心功能。

如果需要恢复 tabBar，在获得图标后：

1. 将4个图标文件放入 `client-user/images/` 目录
2. 在 `app.json` 中恢复 tabBar 配置：

```json
"tabBar": {
  "color": "#666666",
  "selectedColor": "#1989fa",
  "backgroundColor": "#ffffff",
  "borderStyle": "white",
  "list": [
    {
      "pagePath": "pages/index/index",
      "text": "首页",
      "iconPath": "images/home.png",
      "selectedIconPath": "images/home-active.png"
    },
    {
      "pagePath": "pages/order-list/order-list",
      "text": "我的订单",
      "iconPath": "images/order.png",
      "selectedIconPath": "images/order-active.png"
    }
  ]
}
```
