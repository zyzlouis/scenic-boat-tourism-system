# 测试数据导入说明

## 导入步骤

### 1. 打开云开发控制台
在微信开发者工具中，点击"云开发"按钮，进入云开发控制台。

### 2. 导入数据

#### 方法一：通过控制台手动导入（推荐）

1. **导入船型数据（boatTypes）**
   - 点击"数据库" → 选择 `boatTypes` 集合
   - 点击"导入" → 选择 `boatTypes.json`
   - 冲突处理模式：选择"插入"
   - 点击"确定"

2. **导入计费配置（pricingConfigs）**
   - 选择 `pricingConfigs` 集合
   - 点击"导入" → 选择 `pricingConfigs.json`
   - 冲突处理模式：选择"插入"

3. **导入船只数据（boats）**
   - 选择 `boats` 集合
   - 点击"导入" → 选择 `boats.json`
   - 冲突处理模式：选择"插入"

4. **导入员工数据（staff）**
   - 选择 `staff` 集合
   - 点击"导入" → 选择 `staff.json`
   - 冲突处理模式：选择"插入"

#### 方法二：通过云函数导入（适合批量导入）

创建一个临时云函数用于导入数据（不推荐新手使用）。

### 3. 验证数据导入成功

在云开发控制台的数据库中：
- `boatTypes` 应该有 3 条记录
- `pricingConfigs` 应该有 3 条记录
- `boats` 应该有 5 条记录
- `staff` 应该有 3 条记录
