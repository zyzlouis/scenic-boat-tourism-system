# 船型图片要求

## 需要的图片

需要准备 **3张** 船型图片：

### 1. 双人船图片
- **文件名**：`double.jpg` 或 `double.png`
- **用途**：双人船展示图

### 2. 四人船图片
- **文件名**：`four.jpg` 或 `four.png`
- **用途**：四人船展示图

### 3. 六人船图片
- **文件名**：`six.jpg` 或 `six.png`
- **用途**：六人船展示图

---

## 图片规格

### 尺寸要求
- **推荐尺寸**：750px × 500px（宽×高）
- **最小尺寸**：600px × 400px
- **最大尺寸**：1000px × 667px
- **宽高比**：3:2 或相近比例

### 格式要求
- **格式**：JPG 或 PNG
- **大小**：每张图片不超过 500KB
- **压缩**：建议使用适当压缩，保证清晰度

### 内容要求
- **主体**：船只应占据画面主要位置（60%-80%）
- **背景**：
  - 建议：湖泊、河流、景区水域等自然背景
  - 或：纯色/渐变背景（白色、浅蓝色等）
  - 避免：杂乱背景
- **光线**：明亮清晰，避免逆光或过暗
- **角度**：侧面或侧前方45度角最佳

### 风格要求
以下任一风格都可以：

**风格1：实物照片**
- 真实的游船照片
- 可以是在水上或岸边拍摄
- 确保画面干净整洁

**风格2：卡通/插画**

- 卡通风格的船只插画
- 色彩明快，风格统一
- 适合如果没有实物照片

## 上传步骤

### 方式一：通过微信开发者工具上传（推荐）

1. **打开云开发控制台**
   - 在微信开发者工具中点击"云开发"

2. **进入云存储**
   - 点击"云存储"标签

3. **创建目录**
   - 点击"新建文件夹"
   - 文件夹名称：`boat-images`

4. **上传图片**
   - 进入 `boat-images` 文件夹
   - 点击"上传文件"
   - 依次上传：
     - double.jpg
     - four.jpg
     - six.jpg

5. **获取图片URL**
   - 点击图片查看详情
   - 复制"File ID"（格式：`cloud://环境ID.xxxx/boat-images/double.jpg`）

### 方式二：批量上传（如果图片较多）

使用云开发 CLI 工具批量上传（可选，较复杂）

---

## 更新数据库

上传图片后，需要更新数据库中的图片URL：

### 手动更新

1. 在云开发控制台 → 数据库
2. 选择 `boatTypes` 集合
3. 找到对应的记录并编辑
4. 修改 `imageUrl` 字段为实际的图片URL

### 示例数据

```json
{
  "code": "DOUBLE",
  "name": "双人船",
  "imageUrl": "cloud://jingqu-9g4s61eh822f9773.6a69-jingqu-9g4s61eh822f9773-1234567/boat-images/double.jpg"
}
```

## 临时方案（如果暂时没有图片）

### 方案1：使用占位图标
- 显示默认的船只图标
- 或者文字替代

### 方案2：使用免费素材网站
推荐网站：
- **Unsplash**：https://unsplash.com/s/photos/boat
- **Pexels**：https://www.pexels.com/search/boat/
- **Pixabay**：https://pixabay.com/images/search/boat/

搜索关键词：
- "paddle boat"（脚踏船）
- "pedal boat"（踏板船）
- "swan boat"（天鹅船）
- "recreational boat"（游船）

## 参考示例

### 双人船参考
- 适合情侣的小型脚踏船
- 通常是天鹅造型或简约设计
- 颜色：白色、粉色、蓝色等

### 四人船参考
- 适合家庭的中型游船
- 造型稳重，空间较大
- 颜色：蓝色、绿色、黄色等

### 六人船参考
- 适合团队的大型游船
- 造型宽敞，座位多
- 颜色：蓝色、红色等醒目颜色
